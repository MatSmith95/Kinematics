# Kinematics

Kinematics simulation and visualization tools for mechanical linkages and systems.

## Overview

This project provides tools for simulating and analyzing kinematic systems including:
- Forward kinematics (FK)
- Inverse kinematics (IK)
- Linkage mechanism design and visualization
- Motion analysis
